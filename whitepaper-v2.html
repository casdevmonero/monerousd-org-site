MoneroUSD Protocol

A private, overcollateralized USD stablecoin. Built on Monero technology. Self-custodied. Sovereign.

Whitepaper V2 · May 2026 · Protocol v1.2.225

The one-line: Every USDm is backed by at least $1.50 of BTC + XMR under FROST threshold custody. Every transaction uses FCMP++ privacy. Every parameter that matters is governed by bicameral on-chain consensus. Every reserve key is held by a multi-host quorum, never a single operator.

Protocol Core

FCMP++ Privacy live

Anonymity set = the entire UTXO set. No ring decoys. ring_size=1 enforced at consensus.

150% Overcollateral live

Every USDm minted is backed by at least 1.5× its value in real BTC + XMR reserves.

Adaptive Fees live

Fees scale 1× → 100× with reserve utilization. Drain attacks become economically self-defeating.

Adaptive Mining Emission live

Block reward proportional to chain activity. Idle chain → smaller blocks → emission accrual; busy chain → more reward.

RandomX PoW live

Same CPU-friendly PoW as Monero. ASIC-resistant. Two-minute block time.

Local-First Wallet live

Private keys never leave your machine. The wallet IS a peer; no relay servers, no session tokens.

Privacy Infrastructure

PIR Side-Index (Path C) live

SimplePIR over an FCMP++-anchored side-index. Operator can't learn which addresses your wallet is querying.

Light Wallet Server (LWS) live

ECIES-encrypted scope-tag index. Sub-second cross-device restore. Operator sees ciphertext only.

SCOPE_TAG_V1.5 live

Every wrapped + custom token visible after a single LWS query during restore. No second slow scan.

Bulletproofs++ live

Real range-proof soundness for all confidential amounts. 544-byte proofs, hardened against the full A1-A12 attack-surface matrix.

Sender-Side Scope Tags live

Senders embed SCOPE_TAG in tx_extra. New deposits indexed at block time, no operator backfill.

Stealth Subaddresses live

Every receive is a fresh one-time address. No address reuse, no on-chain linkability.

Bridge Custody & Cross-Chain

FROST Threshold Sigs live

t-of-n BIP-340 Schnorr over secp256k1. No single host can move reserves. t = floor(2n/3) + 1.

Multi-Host Distribution live

Day-1 solo bootstrap: VPS hot share + operator device + paper cold share. Grows to community via DKG2-add.

Bicameral Sign Policy live

Each validator independently policy-checks every sign request. Coordinator can ask; only validators decide.

Atomic Swap (zk-v6)

Peer-to-peer trustless cross-chain swaps with Schnorr adaptor signatures. No central operator records user↔mint mappings.

HTLC-Locked Deposits

Buy USDm deposits land in BIP-341 Taproot HTLCs. FROST claims after confirmation; user refunds after timeout.

LP Offer Mesh live

Signed LP offers published on chain via /v1/atomic-swap/offers. Operator-blind — no user↔LP linkage stored.

Programmable Privacy

Dark Contracts (DSOL) live

Privacy-respecting smart contracts. Per-call step + memory limits. Private state via Pedersen commitments.

Commit–Reveal Privacy live

Any @batch entry or private-slot write requires commit + reveal within 16 blocks. No mempool front-running.

DarkVM Step Limits live

1M instructions, 256 KB memory cap per call. Deterministic execution, no I/O, no Math.random().

MoneroUSD IDE live

Browser-based DSOL editor + compile + run + deploy. No private keys in the IDE; provider-driven approval.

Nullifier Uniqueness live

Every spend produces a unique nullifier. Double-spend physically impossible at the indexer.

Bytecode Replace (DC_REPLACE) live

Validator-governed in-place upgrade. Contract id stays; bytecode rotates only after consensus vote.

Sovereignty & Censorship Resistance

Sovereign Web Hosting live

Every official frontend mirrored on chain via SITE_PUBLISH. Wallet serves byte-identical bundles locally.

Three-Tier Serving live

Bundled (T1) → chain-anchored (T2) → dev (T3). Always-on UI even with DNS dark.

SW-1..SW-20 Invariants live

Served bytes hash-verified against chain anchor. Loopback-only sovereign server. Provider sandboxed for unverified sites.

SITE_PUBLISH Fees → Reserve live

100% of registration + update fees route to the protocol reserve. Every publish strengthens the peg.

Trusted Node Architecture live

One-click local USDmd daemon. CakeWallet model: local wallet-rpc, optional remote daemon, no relay.

Sovereign Binary Update

Wallet upgrades served from monerousd-installers sovereign site. SHA256 chain-anchored.

Self-Upgrading Ecosystem

On-Chain Constitution live

Frozen-forever slots (founder pay, atomic unit) + amendable-within-bounds parameters. No admin key.

Bicameral Governance live

Every enactment needs BOTH validator quorum AND miner-hashpower quorum. Per-namespace participation floors.

Six Proposal Namespaces live

Idea / Mimir / Upgrade / DSOLUpgrade / Rollback / Constitutional / EmergencyCancel. Each with its own quorum + TTL + bond.

Sybil-Resistant Bonds live

Validator bond scales quadratically with set size. 1000 Sybils cost ~333× min bond.

EmergencyCancel (1-day) live

Community-driven 1-day veto on pending malicious proposals. ⌈n/2⌉ quorum, no miner gate.

DaemonFork (60-day activation)

Consensus binary upgrades need 4n/5 validator + 2n/3 miner consent with 60-day activation window.

Ion Swap DEX & DeFi

Batch AMM live

Commit–reveal batch settlement. Orders encrypted in-mempool. No MEV, no sandwich attacks.

Confidential LP live

LP positions stored as Pedersen commitments. Amounts are private; nullifiers prevent double-claim.

16 Wrapped Assets live

wBTC, wXMR, wETH, wLTC, wDOGE, wBCH, wSOL, wADA, wZEC, wBNB, wUSDT-TRX, wRLUSD & more.

Dark Pool Orders live

Off-chain encrypted limit-order book. Orders revealed only on fill. Order intent is private.

Lending Protocol live

Overcollateralized lending against USDm + wrapped assets. Reserve-aware liquidation curves.

NFT Marketplace live

Private NFT mint + transfer + sale. Royalties enforced at settlement time, not by social convention.
